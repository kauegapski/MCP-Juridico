# Índice Geral do Model Context Protocol (MCP)

Este documento apresenta o **mapa estruturado de navegação** do repositório **MCP-Juridico**, permitindo a localização rápida das fontes normativas, técnicas e institucionais utilizadas como contexto controlado em análises jurídicas, administrativas e técnicas.

A utilização deste índice deve observar obrigatoriamente o disposto no **PROTOCOLO_DE_USO_MCP.md**.

---

## 1. Protocolo de Governança do MCP

- 📄 **PROTOCOLO_DE_USO_MCP.md**  
  Documento central que estabelece:
  - regras de uso do MCP;
  - hierarquia das fontes;
  - limites operacionais do modelo de linguagem;
  - vedação de inferências normativas indevidas.

---

## 2. Legislação Federal

Pasta destinada ao armazenamento de **leis federais e atos normativos primários**, com caráter vinculante.

📁 **LEGISLACAO/**

- Lei nº 13.709/2018 – Lei Geral de Proteção de Dados Pessoais (LGPD)  
- Instruções Normativas federais correlatas

## Observação Final

Este índice não cria obrigações normativas, servindo exclusivamente como **instrumento de navegação e organização lógica** do Model Context Protocol (MCP).

Em caso de dúvida sobre interpretação ou hierarquia das fontes, **prevalece o PROTOCOLO_DE_USO_MCP.md**.
